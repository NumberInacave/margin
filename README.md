# Margin
Minimal Online Markdown Editor

https://numberinacave.github.io/margin/

## Details
Supports many modern Markdown syntax features<br>
Uses localStorage to save your work<br>
Click outside of text area to see preview<br>
Stop moving mouse cursor for 1 second while editing to start focus mode

## Top left controls
☉ - toggle between light and dark mode<br>
⤓ - download document as .md file<br>
𝓲 - view all supported syntax features (toggle)
