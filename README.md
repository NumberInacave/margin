# Margin
Minimal Online Markdown Editor

https://numberinacave.github.io/margin/

## Cmd/ctrl + 
 + b: bold
 + i: italic
 + u: underline
 + s: stroke-through
 + h: header
 + m: medium header

## Details
Made in around 2 hours, inspired by write.as.
Uses localStorage, Download as .md (top left), switch between light and dark mode (top right)
